# Sistema-de-cafeteria-
Sistema de atención para cafetería en Java aplicando POO y estructuras de datos. Uso de Queue (FIFO) para gestionar la cola de clientes y Stack (LIFO) para almacenar el historial de pedidos atendidos, modelando el flujo operativo del negocio. Desarrollo a la simulación de procesos reales con arquitectura modular y control estructurado de datos.
